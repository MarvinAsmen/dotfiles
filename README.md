# Skhd / Yabai

## Applications
| Action        | Key Combination                                                      |
| ------------- | -------------------------------------------------------------------- |

| Restart yabai | <kbd>shift</kbd> + <kbd>lctrl</kbd> + <kbd>lalt</kbd> + <kbd>r</kbd> |
